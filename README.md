🏠 Real Estate Management System

A web-based platform that simplifies property buying, selling, and renting by providing users with an easy-to-use interface to explore and manage real estate listings.

🚀 Features
🔍 Advanced property search (location, price, type)
🏡 Detailed property listings with images & descriptions
👤 User authentication (login/register)
❤️ Wishlist / Save favorite properties
📞 Contact property owners or agents
🛠️ Admin panel for managing users and listings
🛠️ Tech Stack
Layer	Technology
Frontend	HTML, CSS, JavaScript
Backend	Flask / Node.js / Spring Boot
Database	MySQL / MongoDB
Tools	Git, GitHub
📂 Folder Structure

⚙️ Installation
1. Clone the repository
git clone https://github.com/shanmukhamanoj0/RealEstate_mern_Project
cd real-estate-management
2. Install dependencies

For Python (Flask):

pip install -r requirements.txt

For Node.js:

npm install
3. Run the project

For Flask:

python app.py

For Node.js:

npm start
